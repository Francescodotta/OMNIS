## OPENMS INSTALL
To install OpenMS, please follow the instructions on the official OpenMS website: [OpenMS Installation Guide](https://openms.de/install/). This guide provides detailed steps for different operating systems including Windows, macOS, and Linux.
To install it within the docker environment you can use the debian package available at: [OpenMS Releases](https://abibuilder.cs.uni-tuebingen.de/archive/openms/OpenMSInstaller/release/latest/) and then follow the instructions to install the required Qt6 dependencies and to put it within the docker environment.


## THERMO RAW FILE PARSER TOOL 
To install the ThermoRawFileParser tool, you can follow the steps explained in the github repository: [ThermoRawFileParser GitHub](https://github.com/compomics/ThermoRawFileParser/releases). Download the latest release suitable for your operating system and follow the installation instructions provided there.
